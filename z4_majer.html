Zadanie 4 - Teoria JavaScript

Pytania i odpowiedzi

1) Kiedy powstał JavaScript (podaj datę)?
JavaScript powstał w 1995 roku, a w grudniu 1995 został zaimplementowany w przeglądarce Netscape Navigator.

2) Dlaczego jest to język skryptowy?
Ponieważ jest interpretowany – nie trzeba go kompilować do kodu maszynowego, wystarczy przeglądarka aby go uruchomić.

3) Gdzie jest wykonywany JS (klient lub serwer)?
JavaScript działa głównie po stronie klienta (w przeglądarce), ale dzięki Node.js może być także uruchamiany po stronie serwera.

4) Dlaczego JS to bezpieczne narzędzie?
Ponieważ ma ograniczony dostęp do zasobów komputera – nie może swobodnie ingerować w pliki i system użytkownika.

5) Czy jest to język obiektowy, jeśli tak to dlaczego?
Tak, ponieważ wykorzystuje obiekty, metody i zdarzenia do opisu właściwości i zachowań elementów.

6) Wymień zalety JS:
Zaleta 1: Prostota i łatwość nauki.
Zaleta 2: Tworzenie dynamicznych i interaktywnych stron.
Zaleta 3: Odciążenie serwera i mniejsze przesyłanie danych.

7) Framework to:
Zestaw narzędzi/program ułatwiający tworzenie aplikacji w JS (np. React, Angular, Vue).

8) Do aplikacji mobilnych można użyć Framework:
React Native.

9) Aplikacje responsywne to:
Strony/aplikacje, które dostosowują się do rozmiaru ekranu – zmieniają układ i skalują elementy, aby wyglądały dobrze na każdym urządzeniu.

10) Przepisz linie kodu i wytłumacz:
‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›
Tworzy przycisk z napisem „Nowa Strona”. Po kliknięciu uruchamia funkcję WinOpen(), która otwiera nowe okno/stronę.

11) Przepisz linie kodu i wytłumacz:
window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
Otwiera nowe okno z plikiem obraz.html.
- toolbar=no → ukrywa pasek narzędzi
- directories=no → ukrywa przyciski katalogów
- menubar=no → ukrywa menu
- height=280, width=160 → ustawia rozmiary okna
- top=200, left=200 → ustawia pozycję okna na ekranie

12) Opisz instrukcję window.close():
Zamyka bieżące okno przeglądarki otwarte przez skrypt JavaScript.
